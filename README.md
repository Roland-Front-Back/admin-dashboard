# 📊 Admin Dashboard

A responsive and visually appealing Admin Dashboard built using HTML and CSS, inspired by modern UI/UX design practices. This project showcases a structured layout, custom components, and reusable styles for managing user interface elements such as sidebar navigation, header controls, announcements, trending users, and a project showcase.

## 🖥️ Live Preview

Live preview - https://roland-front-back.github.io/admin-dashboard/

## 📌 Features

- 🔍 Search and Notification Bar

- 👤 User Profile with Greetings Section

- 📁 Project Cards with Action Buttons

- 📢 Announcement Feed

- 📈 Trending User List

- 🧭 Sidebar Navigation

- ✨ Responsive Design for different screen sizes

## 📂 Project Structure

```sh
admin-dashboard/
│
├── index.html          # Main HTML file
├── styles.css          # All CSS styles
├── icons/              # Icon images (sidebar, buttons, etc.)
├── colored-icons/      # Profile and trending avatars
└── README.md           # Project documentation
```

## 🚀 Getting Started

To view or modify this project locally:

1. Clone the repository

```sh
git clone https://github.com/yourusername/admin-dashboard.git
cd admin-dashboard
```

2. Open index.html in your browser

No build tools required. This project is purely static (HTML & CSS only).
